# A little album
A little album base on [Slim framework](https://github.com/slimphp/Slim)  

## Structure
```
cache/
    templates/
logs/
public/
    static/          - static contents
    index.php
src/
    lib/
    middleware/
    models/          - model layer
    thirdparty/
    views/           - view layer
    routes.php       - routes config
templates/
```
